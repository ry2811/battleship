# Battleship Game

A simple Battleship game built with pure linked lists for managing the game board and ship positions.

## Features
- 10x10 game grid
- Pure linked list implementation (no arrays/vectors)
- 4 ships with lengths:
  - 1
  - 2
  - 3
  - 4
- Maximum 20 shots per game
- Multiple themes

## How to Play
1. Open `build/Battleship.exe`
2. Use the mouse to select and shoot cells
3. Press `Space` to switch themes
4. Press `Esc` to return to the menu
5. Press `Enter` to start the game

## Technologies
- C++
- Linked List Data Structure

## Project Goal
-Demonstrates how linked lists can be used to manage a Battleship game board and ship logic without using arrays or vectors.
